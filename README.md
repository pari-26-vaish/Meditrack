# 💊 Meditrack

Meditrack is a web-based medication tracking system that helps users manage their prescriptions, set dosage schedules, and keep track of their medication history — built to make health management simple and organized.

## 🚀 Features
- Add, update, and delete medication records
- Track dosage, frequency, and timing for each medicine
- View medication history and upcoming doses
- Simple, responsive UI for easy daily use
- RESTful APIs for managing medication data

## 🛠️ Tech Stack
**Frontend:**
- HTML5
- CSS3
- JavaScript

**Backend:**
- Java
- Spring Boot
- Spring MVC / Spring Data JPA

**Database:**
- MySQL (or your DB of choice)

**Build Tool:**
- Maven
